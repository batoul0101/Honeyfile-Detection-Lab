# Honeyfile Detection Lab 🛡️

## 🎯 Project Objective
This lab demonstrates a **Blue Team** approach to detecting unauthorized access to sensitive data (Honeyfiles). By simulating a real-world attacker scenario, this project showcases the ability to implement endpoint logging, centralize log analysis, and perform incident investigation within a SOC environment.

![Architecture Diagram](diagrams/Architecture.png)

## 🔍 Key Technologies
* **Endpoint:** Windows 11 with Sysmon
* **SIEM:** Splunk
* **Detection Strategy:** Honeytoken/Honeyfile based detection
